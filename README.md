# Battleship Game (Java)

A console-based implementation of the classic Battleship game developed in Java.

## Features

* 10×10 game board
* Aircraft Carrier, Battleship, Cruiser, Submarine, and Destroyer
* Horizontal and vertical ship placement
* Ship length validation
* Ship overlap prevention
* Adjacent ship detection (including diagonals)
* Input validation and error handling
* Console-based user interaction
* Two-player Battleship gameplay
* Ship placement validation
* Adjacency checking
* Fog of War
* Hit and miss detection
* Ship sinking detection
* Win condition
* Turn-based gameplay
* Coordinate validation


## Technologies Used

* Java
* Arrays
* Loops
* Methods
* Object-Oriented Programming concepts

## How to Run

1. Clone the repository:

```bash
git clone https://github.com/mdharris2003/Battleship-Game.git
```

2. Open the project in your preferred IDE.

3. Compile and run:

```bash
javac Main.java
java Main
```
